Objectifs de la formation : Ui Design / Interface Web Ergonomique

  • À l’issue de cette formation, vous maîtriserez à un niveau plus avancé le processus de création de pages Web ergonomiques et la manipulation de la structure d’un site que vous gérez.
  • Connaître les règles de l’UI design et les fautes de style
  • Se former sur les bases spécifiques de la conception des sites Web.
  • Apprendre à optimiser l’expérience utilisateur et mettre en place des pages plus lisibles.
  • Savoir comment configurer l’architecture d’un site Web (organiser l’information, la mise en page etc.)

Programme de la formation : Ui Design / Interface Web Ergonomique

1. S’initier à la conception d’interfaces Web ergonomiques (UI Design) :

Tout d’abord, avant de se lancer dans la conception et l’élaboration proprement dite des interfaces Web ergonomiques pour une meilleure expérience utilisateur, nos formateurs vont vous transmettre des connaissances de base sur l’ergonomie en tant que concept, ainsi que sur ses critères et son graphisme.

  • Conceptualiser le thème;
  • Repérer les critères de base de l’ergonomie ;
  • Prendre conscience des  normes ISO 9241 et de la conformité aux recommandations W3C ;
  • S’initier sur les aspects liés au graphisme et à l’esthétique de l’UI Design ;
  • Définir les objectifs de l’expérience utilisateur à mettre en place sur son site.

2. S’entraîner à la conception visuelle d’interfaces Web :

À ce niveau de la formation, vous allez découvrir une série entière de détails sur la conception visuelle d’interfaces Web (UI design) et l’habillage spécifique à appliquer lors de l’organisation de chaque page.

  • Comprendre le processus de lecture visuelle de la page-écran (y compris parcours et scan visuel) ;
  • Améliorer la répartition et la hiérarchisation de l’information ;
  • Connaître les standards et les conventions de représentation à considérer ;
  • Manipuler de manière précise les notions particulières, telles que la notion d’habillage dans la conception de site,  de la grille de conception, la notion de « Web safe » (area, color, font, etc.) ;
  • S’occuper de la gestion typographique des textes, des images (format, poids, définition) et des contenus riches (audio, vidéo, animations, etc.) ;
  • Gérer les formulaires et les erreurs d’utilisation ;
  • S’initier sur la problématique Flash (portabilité, accessibilité) ;
  • Se mettre à jour sur les apports de JavaScript, AJAX (bibliothèques jQuery), HTML5 et de ses APIs.

3. Mettre en œuvre les interfaces Web ergonomiques :

  • Assimiler des connaissances facilitant la création des menus ;
  • Savoir relier l’ergonomie et la navigation ;
  • Projeter des maquettes et paramétrer une page Web ;
  • Veiller sur l’évaluation ergonomique.

4. Approfondir la conception des interfaces Web ergonomiques :

  • Concevoir une interface orientée UI design  ;
  • Maîtriser des langages d’une application Web (voire structure de l’information et mise en page) ;
  • Opérer avec des maquettes fonctionnelles.
